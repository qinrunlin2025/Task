## Git学习笔记.md

## 📚 GitHub 核心概念理解

### 什么是 GitHub？

**GitHub** 是一个基于 Git 的代码托管平台，同时也是一个开发者社交网络。

### 主要功能：

- 🔄 版本控制 - 跟踪代码变更历史
- 👥 协作开发 - 支持多人协同工作  
- 🚀 开源平台 - 全球最大的开源代码库
- 📊 个人展示 - 展示技术能力和项目经验

## 🔑 GitHub 核心功能解析

### 1. 仓库（Repository）

项目的基本单位，包含所有文件和版本历史。

### 2. Fork（复刻）

复制别人的仓库到自己的账户。

**使用场景**：

- 想要修改别人的项目
- 为开源项目贡献代码
- 学习他人的代码结构

### 3. Pull Request（拉取请求）

向原项目提交修改建议。

**PR 流程**：

1. Fork 项目
2. 本地修改
3. 提交到自己的仓库
4. 创建 Pull Request
5. 等待审核合并

### 4. Clone（克隆）

将远程仓库下载到本地。

```bash
git clone https://github.com/用户名/仓库名.git
```

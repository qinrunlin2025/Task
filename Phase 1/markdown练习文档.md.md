## 1.markdown的基本语法

### 1.1标题

使用#号标记，可以表示1-6级标题， 随#的个数递增，一级标题字号最大，六级标题字号最小。

#一级标题

##二级标题

###三级标题

要显示出效果就将鼠标放在最后一个“#”与文字中间，按下空格键。

### 1.2字体

星号*与下划线_都可以，单是斜体，双是粗体，三是粗斜体.

*这是斜体*

__这是粗体__

***这是粗斜体***

也可以用快捷键：**加粗Ctrl+B**,*斜体Ctrl+I*。

将鼠标放在被加粗加斜的文字任意处，都可看清楚奏效的“星号*与下划线_”。

### 1.3换行

*直接在一句话后敲两个空格* __貌似不行__<br/>两句话之间加一个空行

在编辑的时候，想让一行文字在显示的时候换行，就在中间加`<br/>`

### 1.4引用

Markdown 中引用通过符号 `>` 来实现。`>` 符号后的空格，可有可无。

> 在引用的区块内，允许换行存在，
>
> 换行并不会终止引用的区块。



如果要结束引用，需要一行空白行，来结束引用的区块。

此外，引用还可以嵌套使用：

> 这是一个引用
>
> > 这是一个引用的引用
> >
> > >这是一个引用的引用的引用

### 1.5链接

[链接名称]（链接地址）

<链接地址>

即是：

[这是小q的主页]（https://github.com/qinrunlin2025/Task/blob/main/README.md）或者<https://github.com/qinrunlin2025/Task/blob/main/README.md>

### 1.6图片

![图片描述，可写可不写，但是中括号要有](图片地址，本地链接或者URL地址。)

比如：

![decoration_Geek.png](C:\Users\Lenovo\Desktop\复刻招新网站要求点\images\decoration_Geek.png)



也可以**修改位置和图片大小**：

！[](图片地址，本地链接或者URL地址#pic_center =长×宽)

![修改后](C:\Users\Lenovo\Desktop\复刻招新网站要求点\images\decoration_Geek.png#pic_center =30×30)

***att***:pic_center,"="前有“ ”（空格），是×不是*

其实自己Ctrl+v粘贴图片更为便捷

将鼠标放在图片任意处，都可看清楚奏效的“链接”。

### 1.12表格

表格使用|来分割不同的单元格，使用-来分隔表头和其他行

* ：-：将表头及单元格内容左对齐
* -：：将表头及单元格内容右对齐
* ：-：：将表头及单元格内容居中
* 实则点开点击表头任意处可以看到**表格右上方竖着三个点**再调整*加减行列*和*左右对齐*

```app
|项目：-：|价格：-|数量-：|
--------|-----： | ：----:
|计算机  |￥817  |  5   |
```

| 项目   | 价格  | 数量 |
| :----- | :---: | ---: |
| 计算机 | ￥817 |    5 |

### 1.13脚注

我[^1]不会[^2]使用表格,可是现在会了。[^T]

[^1]:小q
[^2]:不会没关系
[^T]:好耶好耶

```app
我[^1]不会[^2]用表格，可是现在会了。[^T]
[^1]：小q
[^2]：不会没关系
[^T]：好耶好耶
```

* 脚注自动被搬运到最后面，请到文章末尾查看，
* 并且脚注后方的链接可以直接跳转回到加注的地方。

### 1.14特殊符号

* 反斜线\在英文输入法下在Enter键上面

### 2.5Html

Markdown支持原生HTML语法，可以用Html写一个纵跨两行的表格：

```html
<table>
    <tr>
        <th rowspan="2">值班人员</th>
        <th>星期一</th>
        <th>星期二</th>
        <th>星期三</th>
    </tr>
    <tr>
        <td>李强</td>
        <td>张明</td>
        <td>王平</td>
    </tr>
</table>
```

也可以实现对字体格式的改变

```html
<font face="楷体" color=#00fffff size=5>改变文字格式</font>
```

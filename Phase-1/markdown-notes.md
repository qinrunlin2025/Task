# Markdown学习笔记.md

## 1.标题(Headings)

### 1.1主要语法（#号方式）

-在文字前加"#"号来创建标题

-"#"的数量对应标题级别（1-6级）

-语法："#标题文字"

**示例：**

````markdown
#一级标题

##二级标题

###三级标题

####四级标题

#####五级标题

######六级标题
```
````

### 1.2替代语法（下划线方式）

-在文字下面加"==="表示一级标题

-在文字下面加"---"表示二级标题

-只需要在文字下方一行添加即可

**示例：**

````markdown
这是一级标题
===
这是二级标题
---
```
````

### 1.3最佳实践

#### 空格规则

-√**要这样做**："#标题文字"（#号和文字间有空格）

-×**不要要这样做**："#标题文字"（#号和文字间无空格）

#### 空行规则

-√**要这样做**：在标题前后都留空行

-×**不要要这样做**：标题紧挨着其他文字

**正确示例**:

````markdown
这是前面的段落。

#标题

这是后面的段落。
```
````

### 1.4使用注意

-"#"号和文字之间需要有一个空格

-建议在标题上下留空行，让结构更清晰

## 2.段落(Paragraphs)

### 2.1语法规则

-直接输入文字就会自动成为段落

-段落之间用**空行**进行分隔

-一个空行就表示开始新的段落

### 2.2示例演示

````markdown
这是第一个段落。

这是第二个段落，与第一个段落之间有一个空行。

这是第三个段落。
```
````

### 2.3最佳实践

####  对齐规则

-√**要这样做**：保持文字左对齐，不要缩进

-×**不要要这样做**：在段落前添加空格或制表符

**正确示例**:

````markdown
这是正确的段落格式。
段落之间用空行分隔。

这是新的段落。
```
````

**错误示例**:

````markdown
  这是错误的格式。（前面有空格）
    这是错误的格式。（前面有制表符）
```
````

## 3.换行

### 基本用法

-行尾加**两个空格**再回车，就能换行

-或者用<br> 标签换行

### 示例

```markdown
第一行（后面有两个空格）
第二行

第一行<br>
第二行
```

### 斜体

**斜体文字** 或__斜体文字__

***粗斜体文字***

-推荐用**星号**："**粗体**""**斜体**"

-避免在文字中间用下划线
